NAME
       PPI::Structure::Constructor - Anonymous hash or array constructor

SYNOPSIS
	 my $array = [ 'foo', 'bar' ];
	 my $hash  = { foo => 'bar' };

INHERITANCE
	 PPI::Structure::Constructor
	 isa PPI::Structure
	     isa PPI::Node
		 isa PPI::Element

DESCRIPTION
       "PPI::Structure::Constructor" is the class used for anonymous "ARRAY"
       reference of "HASH" reference constructors.

METHODS
       "PPI::Structure::Constructor" has no methods beyond those provided by
       the standard PPI::Structure, PPI::Node and PPI::Element methods.

       Got any ideas for methods? Submit a report to rt.cpan.org!
