NAME
       MooseX::Meta::TypeCoercion::Structured -
       MooseX::Meta::TypeCoercion::Structured - Coerce structured type
       constraints.

DESCRIPTION
       We need to make sure we can properly coerce the structure elements
       inside a structured type constraint.  However requirements for the best
       way to allow this are still in flux.  For now this class is a
       placeholder.
