NAME
       Moose::Meta::Attribute::Native::MethodProvider::Bool - role providing
       method generators for Bool trait

DESCRIPTION
       This is a role which provides the method generators for
       Moose::Meta::Attribute::Native::Trait::Bool. Please check there for
       documentation on what methods are provided.
