NAME
       DROP_GROUP - remove a database role

SYNOPSIS
       DROP GROUP [ IF EXISTS ] name [, ...]

DESCRIPTION
       DROP GROUP is now an alias for DROP ROLE (DROP_ROLE(7)).

COMPATIBILITY
       There is no DROP GROUP statement in the SQL standard.

SEE ALSO
       DROP ROLE (DROP_ROLE(7))
