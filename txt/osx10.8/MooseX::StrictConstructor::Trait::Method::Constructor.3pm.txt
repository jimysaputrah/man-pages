NAME
       MooseX::StrictConstructor::Trait::Method::Constructor - A role to make
       immutable constructors strict

VERSION
       version 0.16

DESCRIPTION
       This role simply wraps "_generate_BUILDALL()" (from
       "Moose::Meta::Method::Constructor") so that immutable classes have a
       strict constructor.
