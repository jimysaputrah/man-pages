NAME
       Net::DNS::Resolver::Cygwin - Cygwin Resolver Class

SYNOPSIS
	use Net::DNS::Resolver;

DESCRIPTION
       This class implements the cygwin specific portions of
       "Net::DNS::Resolver".

       No user serviceable parts inside, see Net::DNS::Resolver for all your
       resolving needs.
