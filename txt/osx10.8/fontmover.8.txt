NAME
     fontmover -- Mac OS X system font mover

SYNOPSIS
     fontmover -d

DESCRIPTION
     fontmover is a helper tool that moves fonts around on behalf of the font
     server.

     Users should not run fontmover manually.

SEE ALSO
     fontd(8)

HISTORY
     The fontmover tool first appeared in MacOS X 10.6.
