NAME
     activitymonitord -- Activity Monitor daemon

SYNOPSIS
     activitymonitord

DESCRIPTION
     activitymonitord provides services for the Activity Monitor application.

     There are no configuration options to activitymonitord.  Users should not
     run activitymonitord manually.
