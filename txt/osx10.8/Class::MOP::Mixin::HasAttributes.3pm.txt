NAME
       Class::MOP::Mixin::HasAttributes - Methods for metaclasses which have
       attributes

VERSION
       version 2.0205

DESCRIPTION
       This class implements methods for metaclasses which have attributes
       (Class::MOP::Class and Moose::Meta::Role). See Class::MOP::Class for
       API details.
