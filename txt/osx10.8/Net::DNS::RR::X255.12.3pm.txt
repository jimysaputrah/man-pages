NAME
       Net::DNS::RR::X25 - DNS X25 resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS X25 resource records.

METHODS
   psdn
	   print "psdn = ", $rr->psdn, "\n";

       Returns the PSDN address.
