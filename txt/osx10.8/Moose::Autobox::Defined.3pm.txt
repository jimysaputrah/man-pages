NAME
       Moose::Autobox::Defined - the Defined role

SYNOPOSIS
	 use Moose::Autobox;

	 my $x;
	 $x->defined; # false

	 $x = 10;
	 $x->defined; # true

DESCRIPTION
       This is a role to describes a defined value.

METHODS
       defined
       meta
