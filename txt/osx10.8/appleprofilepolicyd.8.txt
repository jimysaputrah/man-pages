NAME
     appleprofilepolicyd -- AppleProfileFamily access control daemon

SYNOPSIS
     appleprofilepolicyd

DESCRIPTION
     appleprofilepolicyd is a system daemon that implements a policy for the
     services provided by AppleProfileFamily.  When an AppleProfileFamily user
     asks to perform an action, this daemon is invoked to make the final pol-
     icy decision.
