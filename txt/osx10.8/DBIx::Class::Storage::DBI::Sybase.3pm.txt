NAME
       DBIx::Class::Storage::DBI::Sybase - Base class for drivers using
       DBD::Sybase

DESCRIPTION
       This is the base class/dispatcher for Storage's designed to work with
       DBD::Sybase

METHODS
   using_freetds
       Whether or not DBD::Sybase was compiled against FreeTDS. If false, it
       means the Sybase OpenClient libraries were used.
