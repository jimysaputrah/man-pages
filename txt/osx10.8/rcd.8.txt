NAME
     rcd -- Remote control daemon.

SYNOPSIS
     rcd

DESCRIPTION
     rcd routes media transport key and remote controller commands to the
     appropriate applications.	It is a daemon that is launched when these
     commands are received.  It is not meant to be invoked directly.
