NAME
       App::Packer::PAR - Pack applications in a single executable file

DESCRIPTION
       This module is a modified version of App::Packer, temporarily shipped
       with PAR until it is merged into newer versions of App::Packer.

SEE ALSO
       See App::Packer for the programming interface.
