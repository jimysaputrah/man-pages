NAME
       glGetFragDataIndex - query the bindings of color indices to
       user-defined varying out variables

C SPECIFICATION
       GLint glGetFragDataIndex(GLuint program, const char * name);

PARAMETERS
       program
	   The name of the program containing varying out variable whose
	   binding to query

       name
	   The name of the user-defined varying out variable whose index to
	   query

DESCRIPTION
       glGetFragDataIndex returns the index of the fragment color to which the
       variable name was bound when the program object program was last
       linked. If name is not a varying out variable of program, or if an
       error occurs, -1 will be returned.

NOTES
       glGetFragDataIndex is available only if the GL version is 3.3 or
       greater.

ERRORS
       GL_INVALID_OPERATION is generated if program is not the name of a
       program object.

SEE ALSO
       glCreateProgram(), glBindFragDataLocation(),
       glBindFragDataLocationIndexed(), glGetFragDataLocation()
