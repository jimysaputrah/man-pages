NAME
       MooseX::Singleton::Role::Meta::Method::Constructor - Constructor method
       role for MooseX::Singleton

VERSION
       version 0.27

DESCRIPTION
       This role overrides the generated object "new" method so that it
       returns the singleton if it already exists.
