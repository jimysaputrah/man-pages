NAME
       DateTime::Locale::sr_ME

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'sr_ME' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Serbian Montenegro.

DATA
       This locale inherits from the DateTime::Locale::sr_Latn_ME locale.

       It contains the following data.
