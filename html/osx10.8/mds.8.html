NAME
     mds -- metadata server

SYNOPSIS
     mds

DESCRIPTION
     mds is the metadata server.  It serves all clients of the metadata APIs,
     including Spotlight.

     There are no configurations to mds, and users should not run mds manu-
     ally.

SEE ALSO
     mdfind(1), mdls(1), mdutil(1)
