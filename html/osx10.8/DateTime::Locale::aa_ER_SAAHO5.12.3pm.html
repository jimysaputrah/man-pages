NAME
       DateTime::Locale::aa_ER_SAAHO

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'aa_ER_SAAHO' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Afar Eritrea Saho.
