NAME
       DateTime::Locale::cch

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'cch' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Atsam.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
