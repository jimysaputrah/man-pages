NAME
       Net::LDAP::Intermediate::SyncInfo - LDAPv3 Sync Info Message object

SYNOPSIS
       See Net::LDAP::Control::SyncRequest

DESCRIPTION
       "Net::LDAP::Intermediate::SyncInfo" provides an interface for the
       creation and manipulation of objects that represent the "Sync Info
       Message" as described by RFC 4533.

METHODS
       As with Net::LDAP::Intermediate each constructor argument described
       above is also avaliable as a method on the object which will return the
       current value for the attribute if called without an argument, and set
       a new value for the attribute if called with an argument.
