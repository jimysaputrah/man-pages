NAME
       DateTime::Locale::cs

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'cs' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Czech.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
