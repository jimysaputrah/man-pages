NAME
       DateTime::Locale::ms_MY

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ms_MY' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Malay Malaysia.

DATA
