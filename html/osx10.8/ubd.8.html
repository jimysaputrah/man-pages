NAME
     ubd -- ubiquity daemon, part of Mobile Documents

SYNOPSIS
     ubd

DESCRIPTION
     ubd is the ubiquity server process. It is primarily used for "Mobile Doc-
     uments".

     There are no configuration options to ubd, and users should not run ubd
     manually.
