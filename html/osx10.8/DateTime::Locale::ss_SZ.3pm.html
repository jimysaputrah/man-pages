NAME
       DateTime::Locale::ss_SZ

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ss_SZ' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Swati Swaziland.

DATA
       This locale inherits from the DateTime::Locale::ss locale.

       It contains the following data.
