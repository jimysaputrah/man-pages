NAME
       Net::DNS::Resolver::UNIX - UNIX Resolver Class

SYNOPSIS
	use Net::DNS::Resolver;

DESCRIPTION
       This class implements the UNIX specific portions of
       "Net::DNS::Resolver".

       No user serviceable parts inside, see Net::DNS::Resolver for all your
       resolving needs.
