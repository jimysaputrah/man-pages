NAME
       DateTime::Locale::trv

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'trv' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Taroko.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
