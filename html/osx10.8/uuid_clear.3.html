NAME
       uuid_clear - reset value of UUID variable to the NULL value

SYNOPSIS
       #include <uuid/uuid.h>

       void uuid_clear(uuid_t uu);

DESCRIPTION
       The uuid_clear function sets the value of the supplied uuid variable uu
       to the NULL value.
