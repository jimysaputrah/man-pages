NAME
       MooseX::Declare::Syntax::Keyword::Clean - Explicit namespace cleanups

DESCRIPTION
       This keyword will inject a call to namespace::clean into its current
       position.

METHODS
   parse
	 Object->parse(Object $context)

       This will inject a call to namespace::clean "-except =" 'meta'> into
       the code at the position of the keyword.
