NAME
       DateTime::Locale::ak_GH

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ak_GH' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Akan Ghana.

DATA
       This locale inherits from the DateTime::Locale::ak locale.
