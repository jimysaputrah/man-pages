NAME
       DateTime::Locale::lv_LV

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'lv_LV' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Latvian Latvia.

DATA
       This locale inherits from the DateTime::Locale::lv locale.

       It contains the following data.
