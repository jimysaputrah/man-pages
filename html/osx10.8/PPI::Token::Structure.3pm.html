NAME
       PPI::Token::Structure - Token class for characters that define code
       structure

INHERITANCE
	 PPI::Token::Structure
	 isa PPI::Token
	     isa PPI::Element

DESCRIPTION
       The "PPI::Token::Structure" class is used for tokens that control the
       generally tree structure or code.

       This consists of seven characters. These are the six brace characters
       from the "round", "curly" and "square" pairs, plus the semi-colon
       statement separator ";".

METHODS
       This class has no methods beyond what is provided by its PPI::Token and
       PPI::Element parent classes.
