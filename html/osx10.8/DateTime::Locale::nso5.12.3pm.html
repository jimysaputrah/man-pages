NAME
       DateTime::Locale::nso

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'nso' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Northern Sotho.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
