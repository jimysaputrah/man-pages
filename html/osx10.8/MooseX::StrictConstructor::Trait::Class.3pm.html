NAME
       MooseX::StrictConstructor::Trait::Class - A role to make immutable
       constructors strict

VERSION
       version 0.16

DESCRIPTION
       This role simply wraps "_inline_BUILDALL()" (from "Moose::Meta::Class")
       so that immutable classes have a strict constructor.
