NAME
     mdcheckschema -- simple mdimporter schema validation tool

SYNOPSIS
     mdcheckschema schema.xml [...]

DESCRIPTION
     The mdcheckschema command validates the mdimporter schema files provided
     as arguments and indicates whether the schema could be successfully
     parsed.

SEE ALSO
     mdfind(1), mdimport(1), mdutil(1), mdls(1)
