NAME
       DateTime::Locale::so_ET

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'so_ET' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Somali Ethiopia.

DATA
       This locale inherits from the DateTime::Locale::so locale.

       It contains the following data.
