NAME
       DateTime::Locale::ve

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 've' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Venda.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
