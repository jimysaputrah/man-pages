NAME
       DBIx::Class::Manual - Index of the Manual

DESCRIPTION
       This is the DBIx::Class users manual. DBIx::Class is a SQL->OOP mapper.
       This means that it can represent your SQL tables as perl classes, and
       give you convenient accessors and methods for retrieving and updating
       information from your SQL database.
