NAME
       doctools::nroff::man_macros  - Default CSS style for NROFF export plug-
       ins

SYNOPSIS
       package require Tcl  8.4

       package require doctools::nroff::man_macros  ?0.1?

       ::doctools::nroff::man_macros::contents

_________________________________________________________________

DESCRIPTION
       This package provides a single command providing access to the  defini-
       tion  of  the  nroff man macro set to use for NROFF markup generated by
       the various NROFF export plugins.

       This is an internal package of doctools, for  use  by  export  plugins,
       i.e.  the  packages  converting	doctools related documented into other
       formats, most notably nroff.

API
       ::doctools::nroff::man_macros::contents
	      This command returns the text of the default CSS	style  to  use
	      for NROFF generated by the various NROFF export plugins.
