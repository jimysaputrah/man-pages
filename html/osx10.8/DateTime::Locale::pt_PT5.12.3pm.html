NAME
       DateTime::Locale::pt_PT

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'pt_PT' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Portuguese Portugal.

DATA
       This locale inherits from the DateTime::Locale::pt locale.
