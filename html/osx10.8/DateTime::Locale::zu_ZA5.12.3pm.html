NAME
       DateTime::Locale::zu_ZA

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'zu_ZA' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Zulu South Africa.

DATA
       This locale inherits from the DateTime::Locale::zu locale.

       It contains the following data.
