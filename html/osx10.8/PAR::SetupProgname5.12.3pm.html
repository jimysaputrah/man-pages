NAME
       PAR::SetupProgname - Setup $ENV{PAR_PROGNAME}

SYNOPSIS
       PAR guts, beware. Check PAR

DESCRIPTION
       Routines to setup the "PAR_PROGNAME" environment variable.  Read the
       "PAR::Environment" manual.

       The "set_progname()" subroutine sets up the "PAR_PROGNAME" environment
       variable

SEE ALSO
       The PAR homepage at <http://par.perl.org>.

       PAR, PAR::Environment
