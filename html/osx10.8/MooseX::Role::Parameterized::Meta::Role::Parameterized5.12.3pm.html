NAME
       MooseX::Role::Parameterized::Meta::Role::Parameterized - metaclass for
       parameterized roles

DESCRIPTION
       This is the metaclass for parameterized roles; that is, parameterizable
       roles with their parameters bound. See
       MooseX::Role::Parameterized::Meta::Trait::Parameterized which has all
       the guts.
