NAME
       pcap_list_datalinks  -  get a list of link-layer header types supported
       by a capture device

SYNOPSIS
       #include 

       int pcap_list_datalinks(pcap_t *p, int **dlt_buf);

DESCRIPTION
       pcap_list_datalinks() is used to get a list of the supported data  link
       types   of   the   interface   associated  with	the  pcap  descriptor.
       pcap_list_datalinks() allocates an array to  hold  the  list  and  sets
       *dlt_buf.   The	caller	is  responsible  for  freeing  the  array with
       pcap_free_datalinks(3PCAP).

RETURN VALUE
       pcap_list_datalinks() returns the number of  data  link	types  in  the
       array  on  success and -1 on failure.  If -1 is returned, pcap_geterr()
       or pcap_perror() may be called with p as an argument to fetch  or  dis-
       play the error text.

SEE ALSO
       pcap(3PCAP), pcap_geterr(3PCAP), pcap_free_datalinks(3PCAP), pcap-link-
       type(7)
