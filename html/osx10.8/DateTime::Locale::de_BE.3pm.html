NAME
       DateTime::Locale::de_BE

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'de_BE' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for German Belgium.

DATA
       This locale inherits from the DateTime::Locale::de locale.
