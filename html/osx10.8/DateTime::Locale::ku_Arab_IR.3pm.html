NAME
       DateTime::Locale::ku_Arab_IR

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ku_Arab_IR' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Kurdish Arabic Iran.
