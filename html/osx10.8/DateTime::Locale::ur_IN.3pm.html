NAME
       DateTime::Locale::ur_IN

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ur_IN' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Urdu India.
