NAME
       DateTime::Locale::bs

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'bs' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Bosnian.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
