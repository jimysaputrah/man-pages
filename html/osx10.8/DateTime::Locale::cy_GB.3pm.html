NAME
       DateTime::Locale::cy_GB

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'cy_GB' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Welsh United Kingdom.

DATA
       This locale inherits from the DateTime::Locale::cy locale.

       It contains the following data.
