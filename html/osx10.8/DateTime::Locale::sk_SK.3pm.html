NAME
       DateTime::Locale::sk_SK

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'sk_SK' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Slovak Slovakia.

DATA
       This locale inherits from the DateTime::Locale::sk locale.

       It contains the following data.
