NAME
     pthread_self -- get the calling thread's ID

SYNOPSIS
     #include <pthread.h>

     pthread_t
     pthread_self(void);

DESCRIPTION
     The pthread_self() function returns the thread ID of the calling thread.

RETURN VALUES
     The pthread_self() function returns the thread ID of the calling thread.

ERRORS
     None.

SEE ALSO
     pthread_create(3), pthread_equal(3)

STANDARDS
     pthread_self() conforms to ISO/IEC 9945-1:1996 (``POSIX.1'').
