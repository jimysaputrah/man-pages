NAME
       DateTime::Locale::kam

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'kam' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Kamba.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
