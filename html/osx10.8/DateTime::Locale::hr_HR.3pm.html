NAME
       DateTime::Locale::hr_HR

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'hr_HR' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Croatian Croatia.

DATA
       This locale inherits from the DateTime::Locale::hr locale.

       It contains the following data.
