NAME
       DateTime::Locale::ku_Latn_TR

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ku_Latn_TR' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Kurdish Latin Turkey.
