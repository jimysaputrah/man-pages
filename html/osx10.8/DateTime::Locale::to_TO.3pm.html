NAME
       DateTime::Locale::to_TO

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'to_TO' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Tonga Tonga.

DATA
       This locale inherits from the DateTime::Locale::to locale.

       It contains the following data.
