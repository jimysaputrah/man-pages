NAME
       Moose::Autobox::Undef - the Undef role

SYNOPOSIS
	 use Moose::Autobox;

	 my $x;
	 $x->defined; # false

DESCRIPTION
       This is a role to describes a undefined value.

METHODS
       defined
       meta
