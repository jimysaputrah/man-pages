NAME
       Class::MOP::Mixin::HasMethods - Methods for metaclasses which have
       methods

DESCRIPTION
       This class implements methods for metaclasses which have methods
       (Class::MOP::Package and Moose::Meta::Role). See Class::MOP::Package
       for API details.
