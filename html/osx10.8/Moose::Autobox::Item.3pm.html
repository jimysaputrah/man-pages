NAME
       Moose::Autobox::Item - the Item role

DESCRIPTION
       This is the root of our role hierarchy.

METHODS
       meta
       dump
	   Calls Data::Dumper::Dumper.

       perl
	   Same as dump. For symmetry with Perl6's .perl method.

	   Like &print with newline.

       print2
