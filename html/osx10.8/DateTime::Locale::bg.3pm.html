NAME
       DateTime::Locale::bg

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'bg' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Bulgarian.
