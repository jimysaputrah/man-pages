NAME
     fgetwc_l, getwc_l, getwchar_l -- get next wide character from input
     stream

LIBRARY
     Standard C Library (libc, -lc)

SYNOPSIS
     #include 
     #include 
     #include 

     wint_t
     fgetwc_l(FILE *stream, locale_t loc);

     wint_t
     getwc_l(FILE *stream, locale_t loc);

     wint_t
     getwchar_l(locale_t loc);

DESCRIPTION
     The fgetwc_l(), getwc_l(), and getwchar_l() functions are extended locale
     versions of the fgetwc(), getwc(), and getwchar() functions, respec-
     tively.  Refer to their manual pages for details.	Also, see xlocale(3)
     for more information about extended locales.

SEE ALSO
     getwc(3), xlocale(3)
