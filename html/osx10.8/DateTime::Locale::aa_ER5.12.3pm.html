NAME
       DateTime::Locale::aa_ER

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'aa_ER' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Afar Eritrea.

DATA
