NAME
       DateTime::Locale::kl

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'kl' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Kalaallisut.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
