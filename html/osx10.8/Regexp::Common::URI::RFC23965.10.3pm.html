NAME
       Regexp::Common::URI::RFC2396 -- Definitions from RFC2396;

SYNOPSIS
	   use Regexp::Common::URI::RFC2396 qw /:ALL/;

DESCRIPTION
       This package exports definitions from RFC2396. It's intended usage is
       for Regexp::Common::URI submodules only. Its interface might change
       without notice.

REFERENCES
       [RFC 2396]
	   Berners-Lee, Tim, Fielding, R., and Masinter, L.: Uniform Resource
	   Identifiers (URI): Generic Syntax. August 1998.
