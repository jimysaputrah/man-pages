NAME
       DateTime::Locale::tr

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'tr' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Turkish.

DATA
       This locale inherits from the DateTime::Locale::root locale.
