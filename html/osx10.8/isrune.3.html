NAME
     isrune -- valid character test

LIBRARY
     Standard C Library (libc, -lc)

SYNOPSIS
     #include 

     int
     isrune(int c);

DESCRIPTION
     The isrune() function tests for any character that is valid in the cur-
     rent character set.  In the ASCII character set, this is equivalent to
     isascii().

RETURN VALUES
     The isrune() function returns zero if the character tests false and
     returns non-zero if the character tests true.

SEE ALSO
     ctype(3), isascii(3), iswrune(3), ascii(7), isalnum_l

HISTORY
     The isrune() function appeared in 4.4BSD.
