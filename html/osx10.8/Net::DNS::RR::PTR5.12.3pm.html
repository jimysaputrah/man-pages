NAME
       Net::DNS::RR::PTR - DNS PTR resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Pointer (PTR) resource records.

METHODS
   ptrdname
	   print "ptrdname = ", $rr->ptrdname, "\n";

       Returns the domain name associated with this record.
