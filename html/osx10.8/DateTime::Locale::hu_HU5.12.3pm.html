NAME
       DateTime::Locale::hu_HU

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'hu_HU' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Hungarian Hungary.

DATA
       This locale inherits from the DateTime::Locale::hu locale.

       It contains the following data.
