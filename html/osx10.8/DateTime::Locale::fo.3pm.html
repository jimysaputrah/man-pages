NAME
       DateTime::Locale::fo

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'fo' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Faroese.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
