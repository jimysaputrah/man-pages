NAME
       Net::DNS::Resolver::Base - Common Resolver Class

SYNOPSIS
	use base qw/Net::DNS::Resolver::Base/;

DESCRIPTION
       This class is the common base class for the different platform sub-
       classes of Net::DNS::Resolver.

       No user serviceable parts inside, see Net::DNS::Resolver for all your
       resolving needs.
