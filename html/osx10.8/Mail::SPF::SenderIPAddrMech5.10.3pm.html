NAME
       Mail::SPF::SenderIPAddrMech - Abstract base class for SPF record
       mechanisms that operate on the SMTP sender's IP address

DESCRIPTION
       Mail::SPF::SenderIPAddrMech is an abstract base class for SPF record
       mechanisms that operate on the SMTP sender's IP address.  It cannot be
       instantiated directly.  Create an instance of a concrete sub-class
       instead.

       See "Constructors" in Mail::SPF::Mech.
