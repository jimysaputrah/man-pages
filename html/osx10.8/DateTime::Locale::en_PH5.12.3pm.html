NAME
       DateTime::Locale::en_PH

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'en_PH' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for English Philippines.
