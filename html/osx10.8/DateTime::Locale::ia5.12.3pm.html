NAME
       DateTime::Locale::ia

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ia' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Interlingua.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
