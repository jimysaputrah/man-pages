NAME
       Moose::Meta::Role::Method - A Moose Method metaclass for Roles

VERSION
       version 2.0205

DESCRIPTION
       This is primarily used to mark methods coming from a role as being
       different. Right now it is nothing but a subclass of
       Moose::Meta::Method.
