NAME
       MooseX::Getopt::Strict - only make options for attrs with the Getopt
       metaclass

DESCRIPTION
       This is an stricter version of "MooseX::Getopt" which only processes
       the attributes if they explicitly set as "Getopt" attributes. All other
       attributes are ignored by the command line handler.
