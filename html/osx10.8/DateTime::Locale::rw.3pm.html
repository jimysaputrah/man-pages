NAME
       DateTime::Locale::rw

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'rw' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Kinyarwanda.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
