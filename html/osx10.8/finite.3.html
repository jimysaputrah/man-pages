NAME
     finite -- returns 1 for finite x, 0 for infinite x and NaNs.

SYNOPSIS
     #include 

     int
     finite(double x);

DESCRIPTION
     The finite() function returns 1 if -infinity < x < +infinity. It returns
     0 if x is +-infinity or a NaN.
