NAME
       pcap_freealldevs - free a list of capture devices

SYNOPSIS
       #include 

       void pcap_freealldevs(pcap_if_t *alldevs);

DESCRIPTION
       pcap_freealldevs()  is  used  to  free  a  list allocated by pcap_find-
       alldevs().
