NAME
       DateTime::Locale::yo_NG

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'yo_NG' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Yoruba Nigeria.

DATA
       This locale inherits from the DateTime::Locale::yo locale.

       It contains the following data.
