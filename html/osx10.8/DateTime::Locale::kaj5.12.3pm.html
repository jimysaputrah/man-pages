NAME
       DateTime::Locale::kaj

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'kaj' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Jju.

DATA
