NAME
       DateTime::Locale::uk

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'uk' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Ukrainian.
