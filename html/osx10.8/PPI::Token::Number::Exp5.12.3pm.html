NAME
       PPI::Token::Number::Exp - Token class for an exponential notation
       number

SYNOPSIS
	 $n = 1.0e-2;
	 $n = 1e+2;

INHERITANCE
	 PPI::Token::Number::Exp
	 isa PPI::Token::Number::Float
	     isa PPI::Token::Number
		 isa PPI::Token
		     isa PPI::Element

DESCRIPTION
       The "PPI::Token::Number::Exp" class is used for tokens that represent
       floating point numbers with exponential notation.

METHODS
   literal
       Return the numeric value of this token.
