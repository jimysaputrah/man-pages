NAME
       MooseX::ClassAttribute::Trait::Role::Composite - A trait that supports
       applying multiple roles at once

VERSION
       version 0.26

DESCRIPTION
       This trait is used to allow the application of multiple roles (one or
       more of which contain class attributes) to a class or role.

BUGS
       See MooseX::ClassAttribute for details.
