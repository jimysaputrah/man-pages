NAME
       DateTime::Locale::sv

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'sv' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Swedish.

DATA
       This locale inherits from the DateTime::Locale::root locale.
