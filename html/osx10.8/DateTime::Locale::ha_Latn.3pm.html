NAME
       DateTime::Locale::ha_Latn

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ha_Latn' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Hausa Latin.

DATA
       This locale inherits from the DateTime::Locale::ha locale.
