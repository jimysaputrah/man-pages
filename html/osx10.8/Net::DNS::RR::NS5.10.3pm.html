NAME
       Net::DNS::RR::NS - DNS NS resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Name Server (NS) resource records.

METHODS
   nsdname
	   print "nsdname = ", $rr->nsdname, "\n";

       Returns the name of the nameserver.
