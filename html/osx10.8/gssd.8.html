NAME
     gssd -- Generic Security Services Daemon

SYNOPSIS
     gssd

DESCRIPTION
     gssd provides kernel access to the Generic Security Services API (GSS-
     API).

     There are no configuration options to gssd.  Users should not run gssd
     manually.
