NAME
       DateTime::Locale::vi

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'vi' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Vietnamese.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
