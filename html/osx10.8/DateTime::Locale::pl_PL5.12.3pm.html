NAME
       DateTime::Locale::pl_PL

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'pl_PL' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Polish Poland.

DATA
       This locale inherits from the DateTime::Locale::pl locale.

       It contains the following data.
