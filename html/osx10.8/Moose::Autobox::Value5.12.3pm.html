NAME
       Moose::Autobox::Value - the Value role

SYNOPOSIS
	 use Moose::Autobox;

	 # execute a sub on the value
	 print 10->do(sub { $_ * $_ }); # prints 100

DESCRIPTION
       This is a role to describes a defined (non-reference) Perl value.

METHODS
       meta
       do (\&block)
