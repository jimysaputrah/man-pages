NAME
       B::Debug - Walk Perl syntax tree, printing debug info about ops

SYNOPSIS
	       perl -MO=Debug[,OPTIONS] foo.pl

DESCRIPTION
       See ext/B/README and the newer B::Concise, B::Terse.

OPTIONS
       With option -exec, walks tree in execute order, otherwise in basic
       order.
