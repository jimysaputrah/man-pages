NAME
     nlcd -- Network Link Conditioner daemon

SYNOPSIS
     nlcd

DESCRIPTION
     nlcd provides services for the Network Link Conditioner.

     There are no configuration options to the nlcd daemon.  One should not
     run nlcd manually.
