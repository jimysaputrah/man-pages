NAME
       DateTime::Locale::et_EE

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'et_EE' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Estonian Estonia.

DATA
       This locale inherits from the DateTime::Locale::et locale.

       It contains the following data.
