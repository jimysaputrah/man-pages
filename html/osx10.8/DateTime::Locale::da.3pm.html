NAME
       DateTime::Locale::da

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'da' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Danish.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
