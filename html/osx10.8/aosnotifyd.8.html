NAME
     aosnotifyd -- Apple Online Services notifcation daemon

SYNOPSIS
     aosnotifyd

DESCRIPTION
     aosnotifyd AOSNotification notification daemon for Push and other notifi-
     cation-based services.

     There are no configuration options to aosnotifyd.	Users should not run
     aosnotifyd manually.
