NAME
       DateTime::Locale::ee_TG

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ee_TG' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Ewe Togo.

DATA
