NAME
     cookied -- The HTTP cookie storage daemon

SYNOPSIS
     cookied

DESCRIPTION
     cookied is a per-user daemon which manages the user's cookie storage.

     There are no configuration options to cookied.  Users should not run
     cookied manually.
