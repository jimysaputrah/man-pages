NAME
       DateTime::Locale::eu

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'eu' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Basque.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
