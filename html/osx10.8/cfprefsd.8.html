NAME
     cfprefsd -- defaults server

SYNOPSIS
     cfprefsd

DESCRIPTION
     cfprefsd provides preferences services for the CFPreferences and
     NSUserDefaults APIs.

     There are no configuration options to cfprefsd manually.
