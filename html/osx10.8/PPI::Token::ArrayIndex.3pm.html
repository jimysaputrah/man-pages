NAME
       PPI::Token::ArrayIndex - Token getting the last index for an array

INHERITANCE
	 PPI::Token::ArrayIndex
	 isa PPI::Token
	     isa PPI::Element

DESCRIPTION
       The "PPI::Token::ArrayIndex" token represents an attempt to get the
       last index of an array, such as $#array.

METHODS
       There are no additional methods beyond those provided by the parent
       PPI::Token and PPI::Element classes.

       Got any ideas for methods? Submit a report to rt.cpan.org!
