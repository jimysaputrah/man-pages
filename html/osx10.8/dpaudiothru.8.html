NAME
     dpaudiothru -- Display Port audio tool.

SYNOPSIS
     dpaudiothru

DESCRIPTION
     dpaudiothru Launched by dpd daemon to allow audio to play from host
     machine to target machine via display port.  dpaudiothru should only be
     run via dpd daemon.

FILES
     /usr/libexec/dpaudiothru				the daemon
     /System/Library/LaunchDaemons/com.apple.dpd.plist	launchd configuration
							plist
