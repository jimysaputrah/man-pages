NAME
     dmesg -- display the system message buffer

SYNOPSIS
     dmesg [-M core] [-N system]

DESCRIPTION
     Dmesg displays the contents of the system message buffer.	This command
     needs to be run as root.

SEE ALSO
     syslogd(8)

HISTORY
     The dmesg command appeared in 4.0BSD.
