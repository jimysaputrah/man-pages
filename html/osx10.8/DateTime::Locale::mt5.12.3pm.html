NAME
       DateTime::Locale::mt

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'mt' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Maltese.

DATA

       It contains the following data.
