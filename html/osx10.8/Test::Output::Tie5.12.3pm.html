NAME
       Test::Output::Tie - module used by Test::Output to tie STDERR and
       STDOUT

DESCRIPTION
       You are probably more interested in reading Test::Output.

       This module is used to tie STDOUT and STDERR in Test::Output.

   Methods
       TIEHANDLE
	   The constructor for the class.

       PRINT
	   This method is called each time STDERR or STDOUT are printed to.

       PRINTF
	   This method is called each time STDERR or STDOUT are printed to
	   with "printf".

       FILENO
       BINMODE
       read
	   This function is used to return all output printed to STDOUT or
	   STDERR.
