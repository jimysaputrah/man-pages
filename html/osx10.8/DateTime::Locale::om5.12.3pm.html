NAME
       DateTime::Locale::om

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'om' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Oromo.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
