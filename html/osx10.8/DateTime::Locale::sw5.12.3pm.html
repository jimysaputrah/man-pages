NAME
       DateTime::Locale::sw

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'sw' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Swahili.

DATA
       This locale inherits from the DateTime::Locale::root locale.
