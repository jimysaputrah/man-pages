NAME
       DateTime::Locale::wal_ET

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'wal_ET' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Walamo Ethiopia.
