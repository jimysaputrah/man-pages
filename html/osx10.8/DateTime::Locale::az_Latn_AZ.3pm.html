NAME
       DateTime::Locale::az_Latn_AZ

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'az_Latn_AZ' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Azerbaijani Latin Azerbaijan.

DATA

       It contains the following data.
