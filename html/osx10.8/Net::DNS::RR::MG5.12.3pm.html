NAME
       Net::DNS::RR::MG - DNS MG resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Mail Group (MG) resource records.

METHODS
   mgmname
	   print "mgmname = ", $rr->mgmname, "\n";

       Returns the RR's mailbox field.
