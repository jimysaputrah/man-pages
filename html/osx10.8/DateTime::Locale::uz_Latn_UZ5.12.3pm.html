NAME
       DateTime::Locale::uz_Latn_UZ

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'uz_Latn_UZ' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Uzbek Latin Uzbekistan.

DATA
