NAME
     docsetinstalld -- Documentation set installer daemon

SYNOPSIS
     docsetinstalld

DESCRIPTION
     docsetinstalld is the daemon responsible for installing documentation
     sets for use by Apple's developer tools.

FILES
     /usr/libexec/docsetinstalld  daemon

SEE ALSO
     docsetutil(1)
