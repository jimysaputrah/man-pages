NAME
       PPI::Statement::End - Content after the __END__ of a module

SYNOPSIS
	 # This is normal content

	 __END__

	 This is part of an PPI::Statement::End statement

	 =pod

	 This is not part of the ::End statement, it's POD

	 =cut

	 This is another PPI::Statement::End statement

INHERITANCE
	 PPI::Statement::End
	 isa PPI::Statement
	     isa PPI::Node
		 isa PPI::Element

DESCRIPTION
       "PPI::Statement::End" is a utility class designed to serve as a
       contained for all of the content after the __END__ tag in a file.

       It doesn't cover the ENTIRE of the __END__ section, and can be
       interspersed with PPI::Token::Pod tokens.

METHODS
       "PPI::Statement::End" has no additional methods beyond the default ones
       provided by PPI::Statement, PPI::Node and PPI::Element.
