NAME
     mp2bug -- mod_perl bug report generator

SYNOPSIS
     mp2bug

DESCRIPTION
     The mp2bug utilty is used to generate a mod_perl bug report template
     which can be filled out and submitted to the mod_perl project.

SEE ALSO
     httpd(8), perl(1)
