NAME
       DateTime::Locale::kfo

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'kfo' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Koro.

DATA
       This locale inherits from the DateTime::Locale::root locale.

       It contains the following data.
