NAME
     productutil -- product archive utility

SYNOPSIS
     productutil

DESCRIPTION
     productutil is used by Apple's developer tools for validating product ar-
     chives. It is not meant to be invoked directly.

SEE ALSO
     productbuild(1), pkgbuild(1)
