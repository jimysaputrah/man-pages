NAME
       DBIx::Class::Storage::DBI::ADO - Support for DBD::ADO

DESCRIPTION
       This class provides a mechanism for discovering and loading a sub-class
       for a specific ADO backend, as well as some workarounds for DBD::ADO.
       It should be transparent to the user.
