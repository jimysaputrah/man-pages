NAME
       DateTime::Locale::gl_ES

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'gl_ES' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Galician Spain.

DATA
