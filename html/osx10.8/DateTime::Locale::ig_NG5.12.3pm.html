NAME
       DateTime::Locale::ig_NG

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'ig_NG' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Igbo Nigeria.

DATA
       This locale inherits from the DateTime::Locale::ig locale.

       It contains the following data.
