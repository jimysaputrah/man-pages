NAME
       DateTime::Locale::tn_ZA

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'tn_ZA' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Tswana South Africa.

DATA
       This locale inherits from the DateTime::Locale::tn locale.

       It contains the following data.
