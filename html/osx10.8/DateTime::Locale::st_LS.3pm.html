NAME
       DateTime::Locale::st_LS

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'st_LS' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Southern Sotho Lesotho.

DATA
       This locale inherits from the DateTime::Locale::st locale.

       It contains the following data.
