NAME
     revisiond -- storage manager for document revisions

SYNOPSIS
     revisiond

DESCRIPTION
     revisiond is the daemon that manages document revisions created by appli-
     cations and system services.

     There are no configurations to revisiond, and users should not run
     revisiond manually.
