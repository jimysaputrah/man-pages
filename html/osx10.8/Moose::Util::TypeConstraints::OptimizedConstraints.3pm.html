NAME
       Moose::Util::TypeConstraints::OptimizedConstraints - Optimized
       constraint bodies for various moose types

DESCRIPTION
       This file contains the hand optimized versions of Moose type
       constraints, no user serviceable parts inside.

       "HashRef"

       "Object"
       "Role"
