NAME
       Pod::Perldoc::GetOptsOO - Customized option parser for Pod::Perldoc

SYNOPSIS
	   use Pod::Perldoc::GetOptsOO ();

	   Pod::Perldoc::GetOptsOO::getopts( $obj, \@args, $truth )
	      or die "wrong usage";

DESCRIPTION
       Implements a customized option parser used for Pod::Perldoc.
