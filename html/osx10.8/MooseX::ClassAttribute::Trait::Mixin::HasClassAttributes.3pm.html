NAME
       MooseX::ClassAttribute::Trait::Mixin::HasClassAttributes - A mixin
       trait for things which have class attributes

VERSION
       version 0.26

DESCRIPTION
       This trait is like Class::MOP::Mixin::HasAttributes, except that it
       works with class attributes instead of object attributes.

       See MooseX::ClassAttribute::Trait::Class and
       MooseX::ClassAttribute::Trait::Role for API details.

BUGS
       See MooseX::ClassAttribute for details.
