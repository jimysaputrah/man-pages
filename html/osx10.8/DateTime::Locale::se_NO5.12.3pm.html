NAME
       DateTime::Locale::se_NO

SYNOPSIS
	 use DateTime;

	 my $dt = DateTime->now( locale => 'se_NO' );
	 print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Northern Sami Norway.

DATA
       This locale inherits from the DateTime::Locale::se locale.

       It contains the following data.
